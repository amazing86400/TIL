# button CSS

```css
button{
    border: none; // 테두리 없앰
    color: white; // 글자색
    text-align: center;     // 글자 가운데 정렬
    text-decoration: none;  // 글자 효과 제거
    display: inline-block;
    font-size: 1rem;        // 폰트 크기
    border-radius: 10px;    // 버튼 테두리 둥글게
    background-color: #000; // 버튼 배경색
    opacity: 0.9;           // 투명도
    width: 170px;           // 너비
    height: 40px;           // 높이
    cursor: pointer;        // 마우스 갖다대면 커서 포인터로 지정
}
button:hover{  // 버튼에 마우스 갖다대면 색 변하게 만듬
    background-color: rgb(44, 44, 44);
    color: white;
}
```
